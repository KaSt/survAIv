🧠 survaiv

An autonomous AI agent running on a microcontroller, paying for its own LLM inference with x402 micropayments, betting on Polymarket to survive.

What is this?

survaiv is a fully autonomous agent that lives on an ESP32 microcontroller (or in the cloud). It's given $15–25 USDC and must survive by making smart prediction market bets. Every LLM call costs real money via the x402 payment protocol — so the agent must be frugal.

  • Runs on ESP32-C3 (400 KB RAM) or ESP32-S3 (8 MB PSRAM)
  • Pays for GPT/DeepSeek/Llama inference with x402 micropayments ($0.0001–$0.002/call)
  • Bets on Polymarket prediction markets via on-chain CLOB
  • Learns from outcomes — builds trading rules through a "wisdom" system
  • Exports knowledge between boards and cloud instances
  • Web dashboard for live monitoring: P&L, positions, decision log

After flashing

The board creates a Wi-Fi access point named survaiv-XXXX. Connect to it and open http://192.168.4.1 to configure your Wi-Fi, LLM endpoint, and agent settings. Once configured, the agent starts autonomously. Monitor it via the web dashboard on your network.
